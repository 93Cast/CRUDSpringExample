# CRUDSpringExample
🛍 CRUD de Productos – Spring Boot + TailwindCSS

Aplicación web desarrollada con Spring Boot utilizando arquitectura MVC para la gestión de productos.
Permite realizar operaciones CRUD (Crear, Leer, Actualizar y Eliminar) con persistencia en base de datos y una interfaz moderna construida con TailwindCSS.

🚀 Tecnologías utilizadas

Java 17

Maven

Spring Boot

Spring Data JPA

Thymeleaf

H2 Database

TailwindCSS

Lombok

📌 Funcionalidades

Registro de nuevos productos

Listado dinámico de productos

Edición de productos existentes

Eliminación de productos

Persistencia automática con JPA

Interfaz responsiva con TailwindCSS

🏗 Arquitectura

El proyecto sigue el patrón MVC:

Controller

Service

Repository

Model (Entity)

Templates (Thymeleaf)
